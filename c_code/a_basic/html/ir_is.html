웹기초

이미지로 내용을 대처하는 IR, 여러 이미지를 하나로 사용하는 IS기법

IR기법

  • 내용을 읽지 않고 강제로 삭제
  • 내용을 읽지 않고 강제로 삭제
  • 글씨를 들여쓰기 사용방법
  • 영역을 감싸 처리하는 방법
  • 영역을 감싸 처리하는 방법쉽게 복합처리가 가능

IS기법 이해하기

조각난 이미지 파일들을 하나의 파일로 병합 후 배경으로 처리해서 웹 문서 전송속도를 높이는 기법

  • 1,1 이미지
  • 1,3 이미지
  • 2,1 이미지
  • 3,3 이미지
  • 3,5 이미지
  • 2,5 이미지
  • 4,5 이미지
  • 5,6 이미지
